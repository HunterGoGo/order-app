# 이미지 파일 위치

이 폴더에 커피 메뉴 이미지를 넣어주세요.

## 권장 파일명

- `americano-ice.jpg` 또는 `americano-ice.png` - 아메리카노(ICE)
- `americano-hot.jpg` 또는 `americano-hot.png` - 아메리카노(HOT)
- `cafe-latte.jpg` 또는 `cafe-latte.png` - 카페라떼
- `cappuccino.jpg` 또는 `cappuccino.png` - 카푸치노
- `caramel-macchiato.jpg` 또는 `caramel-macchiato.png` - 카라멜 마키아토
- `vanilla-latte.jpg` 또는 `vanilla-latte.png` - 바닐라 라떼

## 사용 방법

이미지를 이 폴더에 넣은 후, 데이터베이스의 `menus` 테이블에서 `image_url` 필드를 업데이트하세요.

예: `/images/americano-ice.jpg`
